<h1>Pokedex-Guide</h1>

Bu proje Pokedex-Guide projesidir. Projemde PokeApi tarafından alınan veri Html ve Css ile oluşturduğum cart içerisine aktardım. Pokemonları isimlerine göre arama yapabillir ve filtreleyebiliriz.
